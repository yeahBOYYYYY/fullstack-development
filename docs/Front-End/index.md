# Front-End 101

Front-end is all the client-side infrastructure, including the design of the site, and it's behavior.

The front-end is made up of three core languages,

- [HTML](./HTML/index.md)
- [CSS](./CSS/index.md)
- [JavaScript](./JavaScript/index.md)

On top of those languages, people made frameworks to make the coding in those languages (or environments) easier,

- [Tailwind CSS](./TailwindCSS/index.md)
