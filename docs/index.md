# Full-Stack 101

## What is it?

Full-stack web development refers to the practice of working on both the **[front-end](./Front-End/index.md)** (client-side) and **back-end** (server-side) portions of an application.

## Roadmap to learning it

I use [this roadmap](https://roadmap.sh/full-stack)  to learn it,

- [CSS](./Front-End/CSS/index.md)
- [JavaScript](./Front-End/JavaScript/index.md)
- [NPM](./NPM/index.md)
- [Tailwind CSS](./Front-End/TailwindCSS/index.md)
- React
